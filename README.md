# Responsive Website 🌐

A simple **responsive website** built using **HTML and CSS**.
The layout automatically adjusts for **desktop, tablet, and mobile devices** using the **viewport meta tag** and **CSS media queries**.

---

## 🚀 Features

* Responsive design for different screen sizes
* Separate HTML and CSS files
* Mobile-friendly layout
* Navigation bar
* Card-based content section
* Uses CSS Grid and Flexbox

---

## 🛠 Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Media Queries

---

## 📱 Responsive Breakpoints

| Device  | Layout    |
| ------- | --------- |
| Desktop | 3 columns |
| Tablet  | 2 columns |
| Mobile  | 1 column  |
